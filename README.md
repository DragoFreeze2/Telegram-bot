# Telegram Tag-All Bot (Optimized)

This repository contains an optimized Tag-All Telegram bot.

## Deploy to Railway

1. Create a new GitHub repository and push these files.
2. Create a Railway account at https://railway.app.
3. New Project -> Deploy from GitHub -> select your repo.
4. In Railway project settings -> Variables, add:
   - `TOKEN` = your_bot_token_here
5. Deploy. Logs are visible in Railway.
